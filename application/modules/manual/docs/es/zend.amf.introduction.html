Introducción

Zend_Amf provee apoyo para el Formato de Mensajes de ActionScript Action Message Format ( AMF ) de Adobe, que permite la comunicación entre Adobe Flash Player y PHP . Específicamente, proporciona una aplicación para un servidor gateway que tramita las solicitudes enviadas desde Flash Player al servidor, mapeando estos requerimientos al objeto y a sus métodos de clase, como así también a llamadas arbitrarias de comunicación.

Las especificaciones (en ingles) de AMF3 son de libre disponibilidad y sirven como referencia para establecer qué tipos de mensajes pueden ser enviados entre Flash Player y el servidor.
