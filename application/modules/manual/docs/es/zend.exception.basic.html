Basic usage

Zend_Exception is the base class for all exceptions thrown by Zend Framework. This class extends the base Exception class of PHP.

Ejemplo 385. Catch all Zend Framework exceptions

try {
    // your code
} catch (Zend_Exception $e) {
    // do something
}

Ejemplo 386. Catch exceptions thrown by a specific component of Zend Framework

try {
    // your code
} catch (Zend_Db_Exception $e) {
    // do something
}
