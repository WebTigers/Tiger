Introducción

Desde su Página Web, XML-RPC se describe como una "...llamada ha procedimientos remotos usando como protocolo de transmisión el HTTP y como codificación de datos el XML. XML-RPC está diseñado para ser tan simple como sea posible, permitiendo al mismo tiempo estructuras complejas de datos que deben ser transmitidos, procesados y devueltos".

Zend Framework ofrece soporte tanto para el consumo de servcios remotos XML-RPC como para la construcción de nuevos servidores XML-RPC.
