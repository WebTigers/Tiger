Nom

setOptions — Set object state based on provided options.

Synopsis

setOptions( $options );

setOptions()

Takes an argument of either an associative array or Traversable object. Recognized keys are detailed under la section intitulée « Configuration Options », with the following behaviors:

  • The namespaces value will be passed to registerNamespaces().

  • The prefixes value will be passed to registerPrefixes().

  • The fallback_autoloader value will be passed to setFallbackAutoloader().
