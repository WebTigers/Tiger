Courriel HTML

Pour envoyer un courriel au format HTML, définissez le corps du message en utilisant la méthode setBodyHTML() à la place de setBodyText(). Le type de contenu MIME sera automatiquement définit à text/html. Si vous utilisez les formats textes et HTML, un message MIME de type multipart/alternative sera automatiquement généré :

Exemple 573. Envoyer des courriels HTML

$mail = new Zend_Mail();
$mail->setBodyText('Mon texte de test');
$mail->setBodyHtml('Mon texte de test');
$mail->setFrom('somebody@exemple.com', 'Un expéditeur');
$mail->addTo('somebody_else@exemple.com', 'Un destinataire');
$mail->setSubject('Sujet de test');
$mail->send();
