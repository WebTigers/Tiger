Utilisation classique

Zend_Exception est le classe de base concernant toutes les exceptions envoyées par Zend Framework. Cette classe étend la classe PHP Exception.

Exemple 386. Attraper toutes les exceptions du Zend Framework

try {
    // votre code
} catch (Zend_Exception $e) {
    // faire quelque chose
}

Exemple 387. Attraper les exceptions envoyées par un composant spécifique de Zend Framework

try {
    // Votre code
} catch (Zend_Db_Exception $e) {
    // faire quelque chose
}
