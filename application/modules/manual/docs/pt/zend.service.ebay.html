Introduction

Zend_Service_Ebay is a simple group of APIs for using eBay web services.

  • Finding.

In order to use Zend_Service_Ebay, you should already have an eBay Application ID. To get a key and for more information, please visit the eBay Developers Program web site.

Attention

Your Application ID is linked to your eBay identity, so take appropriate measures to keep them private.
