Introdução

Zend_Barcode fornece uma maneira genérica para gerar códigos de barras. O componente Zend_Barcode é dividido em dois sub-componentes: objetos de código de barras e renderizadores. Os objetos permitem que você crie os códigos de barras independentemente do renderizador. O renderizador permite desenhar códigos de barras com base no suporte necessário.
