Zend_Layout: Conclusões

O Zend_Layout é um envoltório muito simples que lhe providencia imediatamente o Zend_View com os benefícios de um Two Step View, dando a você a flexibilidade para criar um design em todo o site que você puder introduzir o conteúdo de sua aplicação.

Se você está olhando atentamente aos exemplos, contudo, você pode sair com a ideia de que a funcionalidade é relativamente limitada: como alterar o título da página, introduzir uma tag opcional de script, ou até mesmo criar uma barra lateral opcional? Estas questões são abordadas com o conceito de uma "Visão Composta" -- e são assunto do próximo capítulo no tutorial que abrange "espaços reservados."
