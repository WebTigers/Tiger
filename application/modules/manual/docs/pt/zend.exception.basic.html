Uso básico

Zend_Exception é a classe base para todas as exceções lançadas pelo Zend Framework. Esta classe estende a parte essencial da classe Exception do PHP.

Exemplo 389. Tratando todas as exceções do Zend Framework

try {
    // seu código
} catch (Zend_Exception $e) {
    // faça alguma coisa
}

Exemplo 390. Tratando as excepções lançadas por um componente específico do Zend Framework

try {
    // seu código
} catch (Zend_Db_Exception $e) {
    // faça alguma coisa
}
